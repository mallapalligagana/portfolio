# M V Gagana — Portfolio

This is a Vite + React + Tailwind CSS portfolio template for M V Gagana.

Install:

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

Vercel Deployment:

1. Connect this repository to Vercel.
2. Vercel will detect the static build.
3. Set the build command to:

```bash
npm run build
```

4. Set the output directory to:

```bash
dist
```

Or deploy locally with the Vercel CLI:

```bash
npm install -g vercel
vercel login
vercel
```
